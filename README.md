## Features
- Browse top 50 android repository
- Preview repository details
- Sort by star and update date
- Offline cache
- Light & Dark mode

## Tools & Tech
- MVVM Design Pattern
- JetPack Component
- ViewModel & Flow
- Hilt Dependency Injection
- Retrofit
- Glide
- Proguard
- Unit Testing
- Github API
